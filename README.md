Portfolio profesional enfocado en presentar experiencia, proyectos destacados y stack tecnico con una interfaz moderna e interactiva.

## Demo

- Paradoxical: https://paradoxical-client.vercel.app/
- Astra Automations: https://astrautomations.vercel.app/

## Secciones principales

- Hero profesional con foco en perfil full-stack.
- Sobre mi con logros destacados e idiomas mejorados.
- Proyectos destacados con imagen clicable y apertura en nueva pestana.
- Skills por categorias.
- Experiencia y formacion.
- Contacto y enlaces clave.

## Proyectos destacados incluidos

### 1) Paradoxical

Wikipedia social de autores donde compiten por el mejor articulo para permanecer en plataforma.

**Stack / etiquetas:**
`React` `Node.js` `Express` `MongoDB` `JWT` `Cloudinary` `Nodemailer` `Stripe API`

### 2) Astra Automations

Landing page para cliente con demo de automatizacion conectada con n8n en newsletter y formulario de contacto, con integración de Calendly.

**Stack / etiquetas:**
`React` `Vite` `JavaScript` `n8n` `Calendly` `Email Automation`

## Tecnologias del portfolio

- Frontend: React, Vite, CSS
- 3D/Visual: three, @react-three/fiber, @react-three/drei
- Herramientas: ESLint

## Instalacion local

```bash
npm install
npm run dev
```

## Build de produccion

```bash
npm run build
npm run preview
```

## Deploy en Vercel

1. Conecta este repositorio en Vercel.
2. Framework preset: `Vite`.
3. Build command: `npm run build`.
4. Output directory: `dist`.
5. Deploy.
